Privacy Policy

Applies to: Tethernet (Chrome) — Last updated: July 2026

Short version: These extensions connect your browser to a server running on your own computer. They do not collect, store, or transmit your data to any external server, company, or third party. Everything stays on your machine.

1. Who We Are

Tethernet is a developer tool published by ByteClub. Contact: mike@byteclub.com

2. What These Extensions Do

Tethernet creates a local WebSocket connection between your Chrome browser and a server process running on your own machine (localhost). This allows a locally-running AI assistant (such as Claude Code) to read page content, take screenshots, inspect network traffic, and interact with web pages at your direction.

The extensions are developer tools. They require manual setup, a locally-running server process, and an explicit consent step before any capability is active.

4. Data We Do NOT Collect

  • No data is sent to ByteClub or any third party.
  • No analytics, crash reporting, or telemetry is built into the extension itself.
  • No data is stored on any remote server.
  • No data persists beyond your browser session except the connection URL and consent state, which are stored in browser local storage on your device only.
  • HttpOnly cookies are never accessible to the extension — this is a browser platform limitation.

6. Consent and Control

Tethernet displays a full onboarding page explaining its capabilities and risks before any functionality is enabled. The user must read the disclosure and check a consent checkbox before the extension activates. Consent can be revoked at any time from the extension popup, which immediately disconnects the session and disables all capabilities.

7. Security Architecture

The WebSocket connection is always ws://localhost:PORT — it never connects to any remote server. The connection port is entered manually by the user and changes each session. All data exchanged stays within the user's own machine. The extension has no hardcoded remote endpoints of any kind.

8. Children's Privacy

These extensions are developer tools not intended for users under 13. We do not knowingly collect any information from children.
