Tethernet

Browser Co-Pilot for Claude Code

What is Tethernet?

Tethernet connects your Chrome browser to Claude Code running on your local machine. It lets Claude read pages, take screenshots, interact with forms, and capture network traffic to help you debug and explore your own sites.

All connections are to localhost only. No data is sent to any external server by this extension.

What Tethernet Can Access

Once connected to a Claude Code session, the following capabilities are active:

📄
Page Content & Screenshots Read page text, DOM structure, and take screenshots of any open tab.
🖨
Script Execution Run JavaScript in a page to inspect element state, read storage, or extract data.
🖱
Page Interaction Click, type, scroll, and fill forms on your behalf when you ask Claude to help with a task.
📡
Network Traffic Capture HTTP requests and responses on demand to debug API calls — including response bodies without opening DevTools.
🔒
Cookies & Authentication State Read non-HttpOnly cookies to help debug authentication and session issues. Note: Chrome does not expose HttpOnly cookies to extensions.
🌐
Tab Management See open tabs, switch between them, and open or close tabs as needed.
⚠

Understand the risks

Tethernet gives Claude direct access to your browser. Websites can contain hidden instructions designed to manipulate AI assistants into taking actions you never intended — a technique known as prompt injection.

A malicious page could attempt to instruct Claude to:

  • Read sensitive content from other open tabs
  • Extract private data, tokens, or credentials from page storage
  • Submit forms or take actions on your accounts
  • Capture and expose authentication cookies
🛡
Use on trusted sites Only connect Tethernet when working on sites you trust. Avoid using it on sites with user-generated content you haven't reviewed.
👁
Review before sensitive actions Always read what Claude is about to do before it interacts with accounts, forms, or anything involving your data.
🔒
Disconnect when idle Tethernet is inactive when disconnected. Disconnect from the popup when you're not actively using it.
